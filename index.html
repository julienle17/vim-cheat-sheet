Move Cursor

h - Move to cursor left

j - Move to cursor bottom

k - Move to cursor top

l - Move to cursor right

H - Move to top screen

M - Move to middle screen

L - Move to bottom screen

e - Move forwards to the end of a word

b - Move backwards to the start of word

w - Move fordwards to the start of word

B - Move backwards to the start of big word

W - Move forwards to the start of big word

gg - Go to the first line

G - Go to the last line

0 - Move to the start of the line

$ - Move to the end of the line

% - Move to matching caractère

^ - Move to first caractère on the line no blank

g_ - Move to last caractère on the line no blank

Ctrl + d - Move forwards a half page

Ctrl + u - Move backwards a half page

Ctrl + f - Move forwards a complète page

Ctrl + b - Move backwards a complète page
